Autor: Felix Mercado       Fecha: 17/04/2021     Tag: Programación Web     Lenguaje: JavaScript

Esto es un poco raro usando Java no sucede lo mismo que usando Kotlin. En tu caso usas Kotlin, y lo que debes realizar es escapar los caracteres $ usando previamente el carácter\: private val userTerminal: String = "\$3b\$10\$ggGtlKob8VOmKDjp4s9cg.YmR8T1OssrvOFoI95VUZGpWJG6ZQTty" Si imprimes el valor de la cadena no tendrás problema ya que será la cadena que deseas: "$3b$10$ggGtlKob8VOmKDjp4s9cg.YmR8T1OssrvOFoI95VUZGpWJG6ZQTty"
